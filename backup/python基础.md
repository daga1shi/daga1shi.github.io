## print
```python
print("test")
```
>[!TIP]
>输入代码时用英文输入法
>""双引号不能丢，其内部内容表示字符串
### 其他
**换行**  \n
```python
print("te\nst")
```
**三引号** '''或""" (内部内容直接换行,显示结果也为换行)
```python
print('''随
便
写
点
什
么''')
```
**单双引号转义** \
```python
print("如果想打\"这种字符串类型的引号就在前面加反斜杠")
```
**字符串连接**
```python
print("这个"+"可以让"+"内容连接在一起")
```
