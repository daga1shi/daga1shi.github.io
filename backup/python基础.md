```python

```

>输入代码时用英文输入法
>""双引号不能丢，其内部内容表示字符串

### 整数与浮点数

数字主要分为两种：整数、浮点数

浮点数，即小数

在python中，使用除法会输出浮点数（eg:2/1的结果为2.0而非2）

此外，print里不能把数字和字符串混用

如需混用，需要在数字前加str()

```python
print('一天吃'+str(8)+'顿'）
```

当然这种情况还是比较适合把8直接放在 ’    ‘ 里，但是如果要输出数字变量的话很可能会忘记这件事
