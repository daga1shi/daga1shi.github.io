### 整数与浮点数

数字主要分为两种：整数、浮点数

浮点数，即小数

在python中，使用除法会输出浮点数（eg:2/1的结果为2.0而非2）

此外，print里不能把数字和字符串混用

如需混用，需要在数字前加str()

```python
print('一天吃'+str(8)+'顿'）
```

当然这种情况还是比较适合把8直接放在 ’    ‘ 里，但是如果要输出数字变量的话很可能会忘记这件事

##  条件语句 if

## 列表

```python
 #定义列表
this_is_a_list=['a','b','c']

#在列表中添加元素
this_is_a_list.append('d')

#在列表中删除元素
this_is_a_list.remove('d')

#求列表的元素数量
len(this_is_a_list)

#通过索引获得某个位置的元素
this_is_a_list[2]
```

> [!TIP]
>
> 1.append 和之前 len 等函数名的运用方法不同，主要格式为 **对象.方法名.(……)**
>
> 2.列表里可以存在不同类型的元素
